Demonstrate. Test. Approve.

When you make a new Pull Request, Probo sets up a copy of your site for you to preview. You can test your changes, see what things look like, and — when you're happy — merge in the changes. Probo keeps the environments around as long as you like, letting your developers, QA testers, project managers, and clients see your work in progress, review your work, or do final User Acceptance Testing.

Optimized for Database-driven Apps

Using your sanitized database backups, each Probo environment has your site's latest content and configuration. No more coordinating with the entire team to schedule a database sync from production to testing environments.

No Time Restrictions

Unlike similar tools that tear down your environment after testing your code, Probo allows you to keep your environment for as long as you need. Probo is the perfect way to allow stakeholders to give feedback on changes prior to merging them into a shared QA environment.

Synchronization

COMING SOON! Probo will offer Synchronization with Top Hosting Environments. Database and file backups are automatically synchronized from Acquia, and soon Pantheon, which will allow for pushing to hosted environments for additional QA.

Optimized

Probo is optimized for Database-driven Apps. Using your sanitized database backups, each Probo environment has your site's latest content and configuration. No more coordinating with the entire team to schedule a database sync from production to testing environments.

Easy integrations with Github, Bitbucket, and Bitbucket Server

We live in pull requests and we bet you do too. Probo creates an environment for each pull request, provides links to the site and test results within the pull request.

For all of your things

You can run anything that can run in a Linux container in Probo. Got an app that uses PHP, Java, Ruby, Javascript, and Python? So do we, and they all run in Probo.

Open Source Software

Built on Docker and written in Node.js, the foundation of Probo is Open Source, and Probo itself has been largely open sourced. We love Open Source, so fork Probo now on Github!

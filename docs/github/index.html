Github Integration

Probo integrates with Github (a popular version control application for open source projects). In the Github pull request user interface, Probo adds a list of sandbox site builds, their automated test status, and their test links. Senior developers who are reviewing the pull request can immediately see whether any automated tests failed against a Probo site build.

For each Probo sandbox site build, click the “Details” link to be directed to an isolated copy of the site, which uses code from the pull request.

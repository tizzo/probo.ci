Site-seeing made easy

Proviso creates a persistent environment to preview and test each development change. Keep the environments as long as you like, letting your developers, QA testers, project managers, and clients see your work in progress.

Github and Stash Integration

We live in pull requests and our bet is you do too. Proviso creates an environment for each pull request, and gives you links to the site and test results within the pull request.

{% image 'proviso-github.png' alt="Proviso screenshot in Github" class="full-width" %}

Configured with YAML

A simple .proviso.yml yaml file in your repository tells Proviso how to set up your site and run your tests.
